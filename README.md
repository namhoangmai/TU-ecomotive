# TU-ecomotive

## Lights

### Structure for car headlamps
1. Housing - plastic or alumnium body, provides mechanical support, and protects components inside.
2. Optical system:
  - Reflector
  - Projector lens
  - Light guides/diffuser
3. Light sources - modern systems use LED modules.
4. Electronic drivers - converts 12 V -> constant current for LEDs.
5. Lens cover.

### Structure for rear lamps
1. Housing.
2. Optical system:
   - Light guides or reflectors.
   - Red, amber, and clear sections.
3. Light sources - LEDs.
4. Internal electronics: LED drivers, CAN bus modules.

### All modern automotive lights use 12 V
